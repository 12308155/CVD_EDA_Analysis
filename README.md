Performed Exploratory Data Analysis (EDA) on the CAIR-CVD-2025 dataset using Python. Included data cleaning, handling missing values, and visualizing key patterns through histograms, boxplots, and heatmaps. Identified BMI and systolic blood pressure as major risk factors for cardiovascular disease.
